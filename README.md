
# Project Title - Web based Recruitment System

UI Water Corporation




## Features

- Live previews
- Fullscreen mode
- Cross platform
- Beautiful Interface
- Authentication
- Search Filters


## Run Locally

Clone the project

```bash
  git clone https://github.com/olayidecodes/UI-Water-Corporation
```

Go to the project directory

```bash
  cd my-project
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev
```
